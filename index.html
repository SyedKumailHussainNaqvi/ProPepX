ProPepX: A Unified Framework for Interpretable Bidirectional Interaction-Aware Transfer Learning in Joint Residue-Level Protein–Peptide Binding-Site Prediction

Reciprocal protein–peptide reasoning for residue-level molecular recognition, interface discovery, and interpretable binding-site prediction.

Overview

Reciprocal protein–peptide reasoning for residue-level interface discovery

ProPepX explicitly models the reciprocal recognition process between interacting protein and peptide partners. Instead of treating interface prediction as two independent tasks, the framework uses bidirectional cross-attention and gated fusion to learn partner-aware residue representations for both molecular partners.

Unified task Protein-side and peptide-side binding-site prediction in one architecture.
Interaction-aware transfer Pretraining transfers protein–peptide recognition knowledge to low-resource benchmarks.
Biological interpretability Cross-attention maps reveal residue coupling, motifs, and interface hotspots.

Animated overview of the ProPepX prediction workflow.

Method

Bidirectional interaction-aware transfer learning

ProPepX integrates pretrained protein language model embeddings, multi-scale convolutional feature extraction, transformer-based contextual encoding, bidirectional cross-attention, and gated fusion to predict binding residues on both proteins and peptides.

Interpretability

Cross-attention maps reveal biologically meaningful interaction hotspots

ProPepX provides interpretable bidirectional cross-attention maps that expose protein–peptide residue coupling patterns. These maps help identify recognition motifs, interface hotspots, and partner-specific residue contributions underlying binding-site predictions.

Residue coupling Identifies which peptide residues attend to protein interface regions.
Attribution validation Deletion of high-attribution residues reduces predicted binding confidence.
Structural support Predicted hotspots align with structurally meaningful protein–peptide contacts.
