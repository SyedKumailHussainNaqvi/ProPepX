ProPepX A unified framework for interpretable bidirectional interaction-aware transfer learning in joint residue-level protein–peptide binding-site prediction

Overview

Reciprocal protein–peptide reasoning for residue-level interface discovery

ProPepX explicitly models reciprocal recognition between interacting protein and peptide partners. Instead of treating interface prediction as two independent tasks, the framework uses bidirectional cross-attention and gated fusion to learn partner-aware residue representations for both molecular partners.

Unified task Protein-side and peptide-side binding-site prediction in one architecture.
Interaction-aware transfer Pretraining transfers protein–peptide recognition knowledge to low-resource benchmarks.
Biological interpretability Cross-attention maps reveal residue coupling, motifs, and interface hotspots.

Architecture

Model design

ProPepX integrates pretrained PLM embeddings, multi-scale contextual encoding, bidirectional cross-attention, and adaptive gated fusion for biologically interpretable protein–peptide binding-site prediction.

01

Input sequences

Protein and peptide amino-acid sequences are encoded using pretrained protein language models.

02

Context encoders

Multi-scale CNN and Transformer modules capture local motifs and long-range sequence dependencies.

03

Bidirectional attention

Protein-to-peptide and peptide-to-protein cross-attention exchange partner-specific interaction signals.

04

Gated fusion

Intrinsic sequence features are adaptively fused with partner-aware cross-interface features.

05

Residue prediction

The model outputs binding probabilities for each residue on both protein and peptide chains.

Proposed model

Interaction-aware transfer learning strategy

ProPepX learns generalizable protein–peptide recognition patterns through supervised pretraining on large-scale binding data and transfers this knowledge to task-specific residue-level prediction benchmarks.

Conceptual interaction-aware transfer learning workflow for ProPepX. Stage 1 learns transferable binding representations from large-scale generic data, while Stage 2 fine-tunes the pretrained model on task-specific protein or peptide binding-site datasets.

Interpretability & Mechanistic Insight

Mechanistic interface evidence from attention, attribution, and perturbation analysis

ProPepX provides biologically interpretable residue-coupling landscapes through bidirectional gated cross-attention. By combining attention maps, attribution profiles, and perturbation analysis, the framework identifies peptide recognition motifs, protein interface hotspots, and residues that directly influence prediction confidence.

Combined interpretability analysis showing gated cross-interface attention, residue-level attribution profiles, and perturbation-based validation of protein–peptide recognition determinants.

Cross-interface attention Bidirectional gated residue-coupling maps reveal protein–peptide interaction hotspots.
Attribution profiles Residue-level importance scores identify biologically relevant binding determinants.
Perturbation validation Systematic residue deletion confirms the mechanistic importance of high-attribution residues.
