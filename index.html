Overview

Reciprocal protein–peptide reasoning for residue-level interface discovery

ProPepX explicitly models the reciprocal recognition process between interacting protein and peptide partners. Instead of treating interface prediction as two independent tasks, the framework uses bidirectional cross-attention and gated fusion to learn partner-aware residue representations for both molecular partners.

Unified task Protein-side and peptide-side binding-site prediction in one architecture.
Interaction-aware transfer Pretraining transfers protein–peptide recognition knowledge to low-resource benchmarks.
Biological interpretability Cross-attention maps reveal residue coupling, motifs, and interface hotspots.

Architecture

Model design

ProPepX integrates pretrained PLM embeddings, multi-scale contextual encoding, bidirectional cross-attention, and adaptive gated fusion for biologically interpretable protein–peptide binding-site prediction.

01

Input sequences

Protein and peptide amino-acid sequences are encoded using pretrained protein language models.

02

Context encoders

Multi-scale CNN and Transformer modules capture local motifs and long-range sequence dependencies.

03

Bidirectional attention

Protein-to-peptide and peptide-to-protein cross-attention exchange partner-specific interaction signals.

04

Gated fusion

Intrinsic sequence features are adaptively fused with partner-aware cross-interface features.

05

Residue prediction

The model outputs binding probabilities for each residue on both protein and peptide chains.

Interpretability

From predictions to mechanistic interface evidence

ProPepX provides cross-interface attention maps, attribution profiles, perturbation analysis, and residue-coupling landscapes. These outputs help identify recognition hotspots and explain how peptide residues support protein-interface predictions and how protein residues support peptide-interface predictions.

  • Cross-interface residue-coupling maps
  • Protein and peptide binding-site probability tracks
  • Attribution-guided hotspot prioritization
  • Perturbation-based validation of residue importance
Structural validation

Docking and molecular dynamics support biological relevance

Docking-guided validation

Predicted interaction hotspots can be used as restraints for information-driven protein–peptide docking, recovering coherent interface geometries.

100 ns molecular dynamics

High-confidence complexes maintain more stable trajectories with reduced fluctuations and stronger interface stabilization.

Energetic ranking

Model confidence aligns with docking score trends, supporting the physical plausibility of predicted interfaces.
